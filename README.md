# Crypto-Election-Data

A financial research dataset examining the impact of U.S. presidential elections on cryptocurrency markets using event study methodology.

## Overview

This repository contains cryptocurrency price data and event study results for the 2016, 2020, and 2024 U.S. presidential elections. The goal is to analyze how major political events influence crypto market behavior by measuring abnormal returns around election dates.

## Repository Contents

- `2016_Election.csv` — Cryptocurrency price data around the 2016 U.S. presidential election
- - `2016_Election_event_study_results.csv` — Event study results for the 2016 election
  - - `2020_Election.csv` — Cryptocurrency price data around the 2020 U.S. presidential election
    - - `2020_Election_event_study_results.csv` — Event study results for the 2020 election
      - - `2024_Election.csv` — Cryptocurrency price data around the 2024 U.S. presidential election
        - - `2024_Election_event_study_results.csv` — Event study results for the 2024 election
          - - `Crypto-Election-Data.xlsx` — Consolidated Excel workbook with all datasets
           
            - ## Methodology
           
            - The analysis follows a standard financial event study framework, computing cumulative abnormal returns (CAR) for major cryptocurrencies around election event windows.
           
            - ## Tools Used
           
            - - Python / Jupyter Notebook
              - - Pandas, NumPy
